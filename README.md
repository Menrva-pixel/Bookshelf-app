
# Bookshelf App

Dicoding Back-End Beginner Submission


## Run Locally

Clone the project

```bash
  git clone https://github.com/Menrva-pixel/Bookshelf-app
```

Go to the project directory

```bash
  cd Bookshelft-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
